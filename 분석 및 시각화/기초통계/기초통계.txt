연속확률분포 : 경우의 수가 무한히 많아졌을 때의 확률 그래프
    평균치에서 선대칭 분포
    정규분포 => N(평균,표준편차^2)
    분산 = (평균-측정값)^2 = 표준편차^2
        -> 제곱하는 이유
            1. 표준편차의 평균은 0으로 수렴
                (x1-m) + (x2-m) + ... + (xn-m) = (x1 + x2 + ... + xn) - nm = nm - nm =0
                                                (x1 + x2 + ... + xn)/n = m 이기 때문

            2. 절대값을 안쓰고 제곱하는 이유
                통계학 : 분산이 커질수록 예측하기 어려움, 평균으로부터 멀어지기 때문
                머신러닝 : 손실함수 -> 미분 사용, 절대값 사용하면 모든 구간에서 미분 불가능(그래프가 곡선이 안되어 꼭지점에서 미분 불가능 문제 생김)

그래프 적분값의 합 = 1
신뢰구간 : 원하는 기준에 대한 분포값
신뢰구간은 m+2시그마 안에 95.4%가 형성됨
신뢰구간 95% 만들고 싶다 -> m+1.96시그마
신뢰구간 99% 만들고 싶다 -> 2.98시그마
정규분포에서 임의의 난수를 뽑았다 = 확률을 반영해서 난수를 뽑겠다는 말

표준화(머신러닝 : 스케일링)
    평균 = 0, 표준편차 = 1인 정규분포로 바꿔주는 것
    정규분포를 (x-m)/시그마 공식에 대입해서 만들어줌 => (측정값 - 평균) / 표준편차
    모든 분포를 통일화시켜 요인들을 비교하기 위해 사용
