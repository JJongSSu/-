의사결정자로서 중개자로서 생각해야 할 가치는?
1. 기능, 가격 : 
    경영 전략 : 원가 절감 전략
    마케팅 : 선구자 마케팅(동일 품질에서는 익숙한 제품을 사용, 시장에 먼저 등장시킴)
    생산관리 : 라인 프로세스
    조직인사 : 분업(한가지 일만), 표준화,단순화

2. 품질
    경영 전략 : 경쟁 우위 전략(가격, 품질)
    마케팅 : 비교 마케팅(선택했을 때 얻을 수 있는 이익을 설명)
    생산관리 : 배치 프로세스, TQM(total quality management : 품질경영 -> 불량 제로화(6시그마))
    조직인사 : 

3. 경제적 이익(프리미엄)
    경영 전략 : 고급화 전략 : 좋은 인재를 육성하겠다는 전략
    마케팅 : 브랜드 마케팅
    생산관리 : 잡샵 프로세스(job-shop), TQM
    조직인사 : HRD
    ex) 일본 샤프/볼펜 : 사용감
        자동차 조수석 손잡이 : 심심하지 않게 만들어 놓은거임
        에르메스 : 15년간 장인 육성

4. 경험과 의미
    고객 경험에 의한 전략
    -> 기본품질, 성능 품질, 감동 품질(기대)
    -> 기대를 만족시켜주었을 때부터 다음에도 그 기대에 부응하지 못하면 실망하게 된다
        (처음에 탕수육 서비스 줬는데 다음에는 안줬을 때 실망하게 되는 심리)
    -> 감동 품질은 한번 제공해주면 그 순간부터 성능 품질로 품질역행을 한다. 
    -> 경쟁력을 갖추기 위해 지속가능한 기술이 필요

    새로운 의미를 부여할 수 있는 상품 개발
    품질은 수치로 표현 가능하지만 경험은 표현하기 어렵다 : 개인마다 니즈가 다르기 때문


품질 중심
    - STP : 고객 세분화 전략
            인구통계(나이...), 지역, 행동, 사상(이과/문과, 진보/보수...)
    - QFD, HoQ : 제공해줄 수 있는 항목을 제시하고 고객들에게 설문 후 결과를 내는 전략
    - SERVQUAL
: 사람마다 다른 욕구를 가지고 경우에 따라 필요성이 다르기 때문에 문제점이 생길 수 있다
: 경험을 알아내고 그에 맞는 제품/서비스를 개발하는 루프가 필요

경험 중심
    고객 중심의 생각 -> 비슷한 경험, 욕구로 분류
    시공간적 배경 + 욕구 + 이유/해결수단 -> 결과 도출 -> 아이디어 상품화
    경험 = 고객의 관심사
